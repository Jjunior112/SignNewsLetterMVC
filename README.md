# 📨 Aplicação CRUD de Inscrição em Newsletter

## 📋 Funcionalidades
- ✅ Inscrição na newsletter
- 🚫 Validação de e-mail
- ⚠️ Mensagens de erro e sucesso
- ❌ Cancelamento de inscrição

## 🔄 Actions Implementadas
- 📩 Subscribe
- 🎊 SuccessSub
- ❓ NotFoundPage
- ⚠️ DuplicateEmail
- ✉️ Unsubscribe
- 🗑️ DeleteConfirmed

## 🚀 Tecnologias Utilizadas
- 💻 C# com ASP.NET MVC
- 🛠️ Entity Framework Core
- 🗄️ SQLite
- 🎨 Bootstrap
